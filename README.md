# Registration Form

This repository contains a simple static HTML registration form.

## Prerequisites

No dependencies or build tools are required. You only need a web browser.

## Project Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/keval-simform/ai-collaboration-demo.git
   ```

2. Move into the project directory:

   ```bash
   cd ai-collaboration-demo
   ```

## Run Locally

You can use either of the following options:

### Option 1: Open the file directly

Open `index.html` in your browser.

### Option 2: Serve the project locally

Run a simple local web server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/index.html` in your browser.

## Project Structure

- `index.html` - registration form page
- `README.md` - project setup and usage instructions
